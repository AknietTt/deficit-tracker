# Месячный дефицит калорий

Трекер похудения: считает норму (TDEE) по формуле Миффлина-Сан Жеора,
задаёт целевой дневной дефицит и суммирует его в месячный итог.
Дни хранятся по календарным месяцам, можно листать историю.

## Запуск локально

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173

## Сборка

```bash
npm run build
```

Собранные файлы окажутся в `dist/` — их можно задеплоить на Vercel,
Netlify, GitHub Pages или любой статический хостинг.

## Хранение данных

Внутри артефактов claude.ai используется встроенное хранилище
(`window.storage`). При запуске как обычный сайт (`npm run dev` /
задеплоенная сборка) `src/storage.js` автоматически переключается на
`localStorage` браузера — данные останутся только в этом браузере на
этом устройстве.
